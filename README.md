# MarAI Front End

## Dependencies
```
Marai Back End
https://github.com/umarkotak/marai-api
default host: http://localhost:9605

Ollama
https://ollama.com/
model: gemma3:latest
default host: http://127.0.0.1:11434
```

## How to run
```
// to install the app
make install

// to run the app
make run

// open app in browser
http://localhost:3000
```
